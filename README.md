# Advanced Calculator

A modern scientific calculator built using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic (+, –, ×, ÷)
- Scientific functions: sin, cos, tan, sqrt, log10, ln, abs
- Supports “sin 30”, “sin30”, and “sin(30)”
- DEG/RAD toggle
- Powers: x², x³, a^b
- Percent support
- Memory functions: M+, MR, MC
- ANS key (use previous result)
- Calculation history (saved in localStorage)
- Keyboard support

## Files
- index.html
- style.css
- script.js

## How to Run
Simply open `index.html` in any browser.

## GitHub Pages Deployment
1. Go to Repository → Settings → Pages  
2. Source: **Deploy from a branch**  
3. Branch: **main**, Folder: **root**  
4. Save  
5. Your site URL will appear after a few seconds.
